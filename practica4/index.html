- Resumen -

El lenguaja de etiquetas de hipertexto o mejor conocido como HTML es el estandar para la creacion de paginas web utilizado en todas y cada una de ellas

y dado que esta relacionado a protolos de envio de este hipertexto por la red, es importante conocer como su sintaxis, etiquetas y diferentes usos, esta practica introduce al tema de creacion de paginas web y su estilo

- Introducción -

HTML se utiliza para crear la estructura básica de una página web, definiendo elementos como encabezados, párrafos, listas, enlaces, imágenes y formularios. Además, permite incorporar contenido multimedia, como videos y audio, mediante etiquetas específicas. La estructura jerárquica de HTML, basada en elementos anidados, proporciona un marco claro para organizar el contenido y facilitar su comprensión por parte de los navegadores web y los motores de búsqueda. El proceso de creación de páginas web con HTML implica escribir código utilizando etiquetas HTML y atributos para definir la estructura y el estilo del contenido. Por ejemplo, para crear un encabezado, se utiliza la etiqueta h1 para indicar el nivel de encabezado y el texto deseado se coloca entre las etiquetas de apertura y cierre. Para agregar una imagen, se utiliza la etiqueta img junto con el atributo src para especificar la URL de la imagen.
